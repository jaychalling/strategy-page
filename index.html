신규 룰 상세 설명

below_price_rule

prev_close (골든10분, 굿모닝): 현재가 ≤ 전일종가 → 매수 차단 + 보유 중이면 전량매도

open_after_0930 (세력감지): 09:30 이후, 현재가 ≤ 당일시가 → 매수 차단 + 보유 중이면 전량매도. 09:30 전에는 비활성.

intensity_floor (체결강도 바닥)

실시간 체결강도(0B field 228)가 임계값 미만으로 N초 연속 유지되면 전량매도.

골든10분: <92 / 15초 · 굿모닝: <97 / 15초 · 세력감지: <92 / 62초

profit_floor (수익 바닥)

수익률이 trigger% 이상 도달한 적 있으면 활성화. 이후 수익률이 floor% 이하로 하락 시 전량매도.

골든10분/굿모닝: 3.5% 도달 → 0.8% 이하 시 매도 · 세력감지: 7.0% 도달 → 4.0% 이하 시 매도

조건부 손절 (after_tp)

1차만 보유 시 손절이 {"after_tp": true, "value": -0.017}이면,

1차 익절(TP1) 실행 전까지 손절 비활성. TP1 체결 후 -1.7% 손절 활성화.

세력감지는 after_tp 없이 -2.9% 무조건 손절.

트레일링 스탑 (pp 방식)

기존: (cur - peak) / peak → 신규: peak_profit - cur_profit ≥ trailing_drop

예: 골든10분 trigger 3.5%, drop 3.1pp → peak에서 +5% 찍었으면, +1.9%로 하락 시 발동 (5.0 - 1.9 = 3.1pp)

지정가 매도 + 타임아웃

매도 시 매도1호가(ask)로 지정가 주문. N초 내 미체결 시 자동 취소 + 시장가 전환.

골든10분: 10초 · 굿모닝: 30초 · 세력감지: 210초

sell_pending 상태 중에는 중복 평가 방지 (HOLD 반환).
